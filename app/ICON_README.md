# Launcher Icons

The launcher icons need to be added to the following directories:
- mipmap-mdpi (48x48)
- mipmap-hdpi (72x72)
- mipmap-xhdpi (96x96)
- mipmap-xxhdpi (144x144)
- mipmap-xxxhdpi (192x192)

For now, you can use Android Studio's Image Asset Studio to generate placeholder icons:
1. Right-click on `res` folder
2. Select New > Image Asset
3. Configure your icon
4. Generate

The icons should be named:
- ic_launcher.png (legacy icon)
- ic_launcher_round.png (round icon for Android 7.1+)
